# rocket_pants-rpm

Provides support for making `RocketPants::Base` work with RPM.

## Installation

Add this line to your application's Gemfile:

    gem 'rocket_pants-rpm'

And then execute:

    $ bundle

And you're done. NewRelic RPM will then work as expected.
